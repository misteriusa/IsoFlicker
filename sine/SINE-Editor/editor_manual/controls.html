Editor controls

When you open the editor, it looks like this:

Menus

At the top of the window, you'll see the menu bar:
  • File: Load/Save controls
    • New: creates a new Preset. The template Preset is 5 minutes long, doesn't loop, and plays a constant 10Hz Isochronic tone at 440Hz
    • Load: edit a .sin file. Note: if the Preset was created manually (for instance, with a text editor), it may not be possible to edit it in this editor
    • Save: saves the Preset to the current file. If no file is loaded, a "save as" prompt appears
    • Save As: saves the Preset to a file chosen by the user
    • Export: Saves the Preset as a regular audio file. Warning: it is NOT possible to edit an exported Preset!
    • Share: Link to the preset sharing platform. See the Sharing section of this manual for more info
    • Import HBX Preset: Imports a Preset for HBX Binaural Player (with some limitations, explained below)
    • Quit: exits the editor
  • Edit: this is for when you make mistakes
    • Undo: undoes the last operation
    • Redo: redoes an undone operation
  • Preset
    • Add Entrainment Track: adds a new Entrainment Track. The new Track plays a constant 10 Hz pulse, with a 440 Hz carrier, at full volume
    • Clone selected Track: clones the currently selected Track. Only Entrainment Tracks can be cloned
    • Delete selected Track: deletes the currently selected Track. Only Entrainment Tracks can be deleted, and you can't delete the last Entrainment Track
    • Set duration: set the new length of the Preset and configure looping
    • Calculate complexity: estimates the complexity of the Preset and tells you what kind of devices may have trouble playing it
    • Optimize: removes useless points and tracks. Minor information loss is possible.
  • ?
    • Help: opens this manual
    • Online tutorial: shows a brief video tutorial
    • About: shows information about the application

Editing envelopes

The Background noise track and all the Entrainment Tracks contain envelopes.
Editing them is easy, but you'll need a mouse.
This is what an envelope might look like:
In this specific case, it's a Background noise envelope.
Each vertical line is a minute. Horizontal lines are there for your convenience.
The red area indicates values that may be too high.
As you can see there are several points: all of them have 2 coordinates: time and value.
If you move the mouse over the graph, it will show the coordinates of that point.
The green line indicates how the value changes over time.
If the mouse is over a Point, its coordinates are shown in blue.
Drag points with the left mouse button to change their coordinates.
Sometimes (like in the image above), the envelope may be very long and go out the screen: to move around, drag the scroll bar under the envelope, and use the mouse wheel to zoom in and out.
Now the envelope fits perfectly, and you can zoom in again if you need more precision.
To add a point, click anywhere in the graph where there isn't a point, and one will be added.
As you can see in the image above, the new point will be created at the time shown next to the mouse cursor, but won't have that value.
To delete a point, right click on it.
As you can see, now we only have one point, which means the envelope is a constant value. It is common to have constant values for the Base Frequency envelopes of Entrainment Track, as they usually don't change note.
Sometimes you may want to manually set the value of a point, you can do this by middle clicking or double clicking a point.

Undo/Redo

If you do something by accident, select Undo from the Edit menu to undo the action.
If you do undo an action by accident, select Redo from the Edit menu to redo the action.

Testing the Preset

Before sharing a Preset, you should test it. You can do it directly in the editor: click Play in the small player at the bottom.
The controls here are self-explanatory: hit Play to start playing the Preset, Pause to pause it, drag the bars to change position and volume, rewind to start from the beginning. When you're done, just pause it.
You can even make changes while the preview is running.
If your Preset has a loop, make sure there's no audible difference between where the start and the end of the looping section.

Importing HBX Binaural Player Presets

HBX Binaural Player is another Brainwave Entrainment application we made, and has quite a lot of Presets.
This editor includes a simple Preset converter: open the File menu, click Import HBX Preset and select the Preset file.
If everything goes fine, you'll see it in the editor.
While importing usually works fine, there are some limitations:
  • Interpolation factors are not supported: they will all be treated as linear (1)
  • The Binaural Frequencies must be between 0 and 40 Hz
  • All the volumes must be between 0 and 1
  • There may be some extra points in some envelopes (they're harmless, but will increase the preset's complexity)
Negative time values are supported.
You may also want to increase the Base Frequency.
